  <action>
Create `js/utils/purchase-simulator.js`:

Two exported functions:

1. `simulateInstallmentPurchase({ totalValue, installments, monthlyInterestRate = 0 })`:
   - Guard: if totalValue <= 0 OR installments <= 0, return null
   - If monthlyInterestRate <= 0 (zero interest): `monthlyPayment = roundMoney(totalValue / installments)`
   - If monthlyInterestRate > 0 (compound interest): PMT formula
     ```
     r = monthlyInterestRate / 100
     factor = Math.pow(1 + r, installments)
     monthlyPayment = roundMoney(totalValue * (r * factor) / (factor - 1))
     ```
   - `totalPaid = roundMoney(monthlyPayment * installments)`
   - `totalInterest = roundMoney(totalPaid - totalValue)`
   - `effectiveRate = totalValue > 0 ? percentage(totalInterest, totalValue) : 0`
   - Return `{ monthlyPayment, totalPaid, totalInterest, installments, effectiveRate }`

2. `computePurchaseImpact({ monthlyPayment = 0, freeBudgetEstimate = 0, estimatedIncome = 0 })`:
   - `newFreeBalance = roundMoney(freeBudgetEstimate - monthlyPayment)`
   - `paymentAsPercentOfFree = freeBudgetEstimate > 0 ? percentage(monthlyPayment, freeBudgetEstimate) : 0`
   - `paymentAsPercentOfIncome = estimatedIncome > 0 ? percentage(monthlyPayment, estimatedIncome) : 0`
   - `viable = freeBudgetEstimate > 0 && monthlyPayment <= freeBudgetEstimate`
   - Return `{ newFreeBalance, paymentAsPercentOfFree, paymentAsPercentOfIncome, viable }`

For helpers (`toNumber`, `roundMoney`, `percentage`): import from `financial-analysis.js` if they were exported in Plan 01 Task 1; otherwise define locally.

Create `tests/phase-01/purchase-simulator.test.js` using `node:test` + `node:assert` covering all behaviors above. Include a test verifying the PMT formula against a known bank calculation: R$1000, 10x, 2% a.m. => monthly ~R$111.33.
